# 运行示例输出

## 安装依赖后运行

```bash
cd crypto-signal-lite
pip install -r requirements.txt
python main.py
```

## 定时模式运行

```bash
python main.py --schedule
```

输出：
```
⏰ 定时任务已启动，每天 UTC 01:00 (北京时间 09:00) 运行
按 Ctrl+C 退出
```

## 图表查看

运行后，打开浏览器访问：
- `charts/arusdt_signal_chart.html` (日线图)
- `charts/arusdt_signal_chart.html` (周线图，会覆盖)

图表包含：
- K线图 + MA20/MA60 均线
- 买入信号（绿色上三角）
- 卖出信号（红色下三角）
- MACD 指标
- RSI 指标
